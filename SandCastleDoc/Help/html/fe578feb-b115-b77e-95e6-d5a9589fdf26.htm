Un DataSet que contiene los datos de la tabla. Si ocurre un error, devuelve null.
Remarks
Este método abre una conexión con la base de datos y ejecuta una consulta SELECT para obtener todos los registros de la tabla especificada. Los resultados se devuelven en un DataSet.
