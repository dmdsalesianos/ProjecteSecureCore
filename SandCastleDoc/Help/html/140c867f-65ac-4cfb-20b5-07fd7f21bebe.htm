Genera una consulta de búsqueda con múltiples criterios y devuelve los resultados en un DataSet.

Parameters

nomTaula  String
Nombre de la tabla en la que buscar.
criteris  DictionaryString, String
Diccionario de criterios de búsqueda, donde la clave es el nombre de la columna y el valor es el valor a buscar.

Return Value

DataSet
Un DataSet con los resultados de la búsqueda. Si ocurre un error, devuelve null.
Remarks
Este método genera dinámicamente una consulta SQL para buscar registros en una tabla específica según los criterios proporcionados. Los nombres de las columnas y valores son incluidos en la consulta de manera segura para prevenir inyecciones SQL. El nombre de la tabla es validado para garantizar que solo contenga caracteres alfanuméricos y guiones bajos.
Example
Ejemplo: Generar una consulta de búsqueda con criterios específicos.
C#
Copy
var criteris = new Dictionary<string, string>
{
    { "Login", "MAND" },
    { "Rol", "Admin" }
};
string nomTaula = "Usuarios";
MantenimentDades manteniment = new MantenimentDades(connectionString);
DataSet resultado = manteniment.GeneraConsultaCerca(nomTaula, criteris);
Esto generará una consulta SQL similar a:
C#
Copy
SELECT * FROM [Usuarios] WHERE 1=1 AND [Login] = 'MAND' AND [Rol] = 'Admin'
Los valores de los parámetros serán configurados con seguridad para prevenir inyección SQL. Ejemplo: Generar una consulta de búsqueda sin criterios.
C#
Copy
string nomTaula = "Usuarios";
MantenimentDades manteniment = new MantenimentDades(connectionString);
DataSet resultado = manteniment.GeneraConsultaCerca(nomTaula, null);
Esto generará una consulta SQL similar a:
C#
Copy
SELECT * FROM [Usuarios] WHERE 1=1
