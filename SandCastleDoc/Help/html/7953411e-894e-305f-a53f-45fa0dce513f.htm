El DataSet que contiene los cambios a aplicar en la base de datos.

El nombre de la tabla en el DataSet.

Return Value

Int32
El número de filas afectadas por la actualización. Devuelve -1 si ocurre un error.
Remarks
Este método realiza una actualización en la base de datos utilizando un DataSet que contiene las modificaciones a aplicar en la tabla especificada.

int result = db.Actualitzar("SELECT * FROM Clientes", ds, "Clientes");
